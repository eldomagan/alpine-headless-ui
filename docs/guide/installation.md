# Installation

## Prerequisites

Alpine Headless UI requires:

- **Alpine.js** v3.0 or higher

## Package Manager

::: code-group

```sh [npm]
npm install alpine-headless-ui
```

```sh [pnpm]
pnpm add alpine-headless-ui
```

```sh [yarn]
yarn add alpine-headless-ui
```

```sh [bun]
bun add alpine-headless-ui
```

:::

## CDN

For quick prototyping, you can use a CDN:

```html
<script defer src="https://cdn.jsdelivr.net/npm/alpine-headless-ui/dist/cdn.min.js"></script>
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3/dist/cdn.min.js"></script>
```
